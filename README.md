# Netflix Analytics Dashboard

An interactive Power BI dashboard created to analyze Netflix movies and TV shows data using visual analytics and KPIs.

## Project Overview

This dashboard provides insights into Netflix content based on:
- Movies vs TV Shows
- Genre distribution
- Ratings analysis
- Content release trends
- Country-wise content availability
- Duration and content growth analysis

## Tools & Technologies Used

- Power BI
- Data Visualization
- Data Cleaning
- Dashboard Design

## Key Features

- Interactive dashboard visuals
- KPI cards for quick insights
- Genre and ratings breakdown
- Global content distribution map
- Trend analysis using charts
